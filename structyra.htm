Общая структура объекта автоматизации

  Посредническое отраслевое предприятие по ведению ВЭД состоит из ряда функциональных подразделений, каждому из которых вменяются определенные функции.

Руководитель посреднического отраслевого предприятия по ведению ВЭД координирует всю работу, отвечает за результаты деятельности, претворяет в жизнь задумки руководства предприятия.

Заместитель по финансам непосредственно подотчетен руководителю предприятия, отвечает за финансовую сторону деятельности посреднического отраслевого предприятия по ведению ВЭД, за его прибыльность.

Заместитель по кадрам организовывает работу подотчетных отделов, принимает непосредственное участие в формировании кадровой структуры объекта автоматизации и решением возникающих социальных вопросов.

Заместитель по маркетингу ответственен за продвижение товара на рынке, защиту деловой репутации предприятия и рекламную деятельность посреднического отраслевого предприятия по ведению ВЭД.

Заместитель по организационным вопросам руководит разрешением возникающих текучих проблем, разработкой новых технологий по продаже продукции и организации различных встреч, акций.

Отдел экономического развития разрабатывает программы по увеличению прибыли, оборачиваемости товаров, программ развития предприятия на кратко- и долгосрочную перспективу.

Отдел финансового планирования занимается разработкой наиболее эффективных методов использования свободных денежных средств,  увеличением поступления активов.

Отдел по социальным вопросам решает текущие социальные вопросы, связанные с деятельностью посреднического отраслевого предприятия по ведению ВЭД, оказывает целенаправленную адресную помощь, руководит благотворительной деятельностью объекта автоматизации.

Отдел кадров занимается поиском и подбором необходимых высококвалифицированных кадров для предприятия, повышением производительности труда.

Отдел по рекламе занимается разработкой рекламной стратегии, формированием имиджа предприятия.

Отдел по маркетинговой стратегии руководит продвижением товара на рынок, разработкой маркетинговых мероприятий по увеличению спроса на продукцию производителя.

Отдел разработки новых технологий осуществляет разработку и внедрение прогрессивных технологий во все стороны жизни предприятия, осуществляет оценку их эффективности и целесообразности использования.

Отдел по организации специальных мероприятий координирует деятельность по организации различных выставок, приемов, встреч и решением возникающих форс-мажорных обстоятельств.
